Zentto Studio

Runtime UI builder as a web component. Generate full applications from JSON — forms, grids, charts, dashboards — in any framework.

GitHub Get Started

Why Zentto Studio?

One JSON config. Any framework. Full application.

📋

30+ Field Types

Text, number, date, select, chips, tree view, signature, address, file upload, charts, and more.

🎨

Visual Designer

Figma-style drag-drop editor with undo/redo, autosave, live preview, and API connection.

⚡

Expression Engine

Safe recursive parser for computed values, visibility rules, and validation — no eval().

🔌

API Integration

Connect to any REST/GraphQL endpoint. Auto-detect fields. Auth built-in.

🏗️

Full App Shell

Sidebar, header, routing, pages, KPI cards, data grids, charts — from one JSON config.

🌐

Framework Agnostic

Vanilla JS, React, Vue, Angular, Blazor, Vite — works everywhere web components work.

Quick Start

Get a form running in under 30 seconds

<!-- Just add one script tag --> <script type="module" src="https://unpkg.com/@zentto/studio@latest/dist/zentto-studio-renderer.js"></script> <!-- Use the web component --> <zentto-studio-renderer id="form"></zentto-studio-renderer> <script> document.getElementById('form').schema = { id: 'demo', version: '1.0', title: 'Contact', layout: { type: 'grid', columns: 2 }, sections: [{ id: 'main', fields: [ { id: 'name', type: 'text', field: 'name', label: 'Name', required: true }, { id: 'email', type: 'email', field: 'email', label: 'Email' }, ]}], actions: [{ id: 'send', type: 'submit', label: 'Send', variant: 'primary' }] }; </script>
